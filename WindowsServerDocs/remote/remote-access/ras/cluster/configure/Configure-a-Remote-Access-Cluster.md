-  步骤1：[使用高级选项部署 ALWAYSON VPN 服务器](../../../vpn/always-on-vpn/deploy/always-on-vpn-adv-options.md)。

-   [步骤2：准备群集服务器](Step-2-Prepare-Cluster-Servers.md)。

-   [步骤3：配置负载平衡群集](Step-3-Configure-a-Load-Balanced-Cluster.md)。
