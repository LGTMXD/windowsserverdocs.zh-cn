# <a name="remap-all-stations"></a>重新映射所有工作站
通过重新映射工作站，可将键盘和鼠标关联到监视器。 重新映射所有工作站时，将清除原始设置（如名称和自动登录信息）。 进行重新映射时，将挂起所有本地用户工作站。  
  
1.  在工作站模式下打开 "MultiPoint 管理器"，然后单击 "**主页**" 选项卡。  
2.  在**任务**之下，单击**重新映射所有工作站**。  
3. 按照工作站屏幕上的说明，将键盘关联到系统中的工作站。
