# <a name="install-updates"></a>安装 更新
建议安装更新（如果有）。 安装更新需要建立 Internet 连接。

1.  从 **“开始”** 屏幕打开 **“控制面板”**。

2.  在 "控制面板" 中，键入 "**更新**"，然后单击 "**检查更新**"。

3.  如果 Windows 更新网站列出了计算机上所需的任何更新，请安装更新。
