# <a name="managing-multipoint-services"></a>管理 MultiPoint 服务
MultiPoint &trade; 服务允许多个用户，每个用户都有自己的独立 Windows 体验，同时共享一台计算机。 用户工作站（由显示器、键盘和鼠标组成），可通过 USB、视频电缆或网络直接连接到主计算机。
