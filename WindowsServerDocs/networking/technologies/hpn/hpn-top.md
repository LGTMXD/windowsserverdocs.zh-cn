# <a name="high-performance-networking-hpn"></a>高性能网络 (HPN) 

 (HPNs 的高性能网络) 在实时数据处理要求中扮演着角色。 例如，数据中心复制、数据中心灾难恢复和高性能分布式计算等活动需要大量数据传输和低网络延迟。 具有动态连接功能的 HPNs 使高性能网络资源更易于访问和管理。
