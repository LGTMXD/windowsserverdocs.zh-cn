| 锁定和限制 | Windows Server 2016 Standard | Windows Server 2016 Datacenter |

| 最大 RAM             | 24 TB           | 24 TB             |

| 服务器可以加入域 | 是            | 是                |
| 边缘网络保护/防火墙 | 否     | 否                 |
| DirectAccess            | 是             | 是                |
| DLNA 解码器和 Web 媒体流 | 是，如果安装为具有桌面体验的服务器 | 是，如果安装为具有桌面体验的服务器 |
