# <a name="gettype"></a>gettype

此**gettype**命令已弃用，并且在将来的 Windows 版本中不保证支持。

此工具包含在 Windows Server 2003 中。 有关详细信息，请参阅[gettype](https://docs.microsoft.com/previous-versions/orphan-topics/ws.10/cc773104(v=ws.10))。
