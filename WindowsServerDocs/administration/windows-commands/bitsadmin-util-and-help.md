# <a name="bitsadmin-util-and-help"></a>bitsadmin util 和帮助

显示**Util**开关的命令行用法。

## <a name="syntax"></a>语法

```
Bitsadmin /Util /Help 
```

## <a name="examples"></a><a name=BKMK_examples></a>示例

以下命令显示**Util**开关的命令行帮助。
```
Bitsadmin /Util /Help
```
