# <a name="bitsadmin-util-and-help"></a>bitsadmin util 和 help

显示 **/util**开关的命令行用法。

## <a name="syntax"></a>语法

```
bitsadmin /util /help
```

## <a name="examples"></a>示例

显示 **/util**开关的命令行帮助：

```
bitsadmin /util /help
```
