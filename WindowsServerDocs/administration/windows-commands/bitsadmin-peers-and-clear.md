# <a name="bitsadmin-peers-and-clear"></a>bitsadmin 对等节点并清除

清除对等列表。

## <a name="syntax"></a>语法

```
bitsadmin /peers /clear
```

## <a name="examples"></a><a name=BKMK_examples></a>示例

下面的示例将清除对等列表。

```
C:\>bitsadmin /peers /clear
```
