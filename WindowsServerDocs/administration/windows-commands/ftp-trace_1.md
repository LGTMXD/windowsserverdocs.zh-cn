# <a name="ftp-trace_1"></a>ftp： trace_1



切换数据包跟踪。

## <a name="syntax"></a>语法

```
trace
```

#### <a name="parameters"></a>参数

none

## <a name="remarks"></a>备注

**Trace**显示运行**ftp**命令时的内部 FTP 函数调用序列。

## <a name="examples"></a>示例

开启和关闭跟踪。
```
trace
```
