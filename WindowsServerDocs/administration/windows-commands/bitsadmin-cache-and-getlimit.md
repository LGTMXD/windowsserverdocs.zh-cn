# <a name="bitsadmin-cache-and-getlimit"></a>bitsadmin 缓存和 getlimit

检索缓存限制。

## <a name="syntax"></a>语法

```
bitsadmin /cache /getlimit
```

## <a name="examples"></a><a name=BKMK_examples></a>示例

下面的示例检索缓存限制。

```
C:\>bitsadmin /cache /getlimit
```
