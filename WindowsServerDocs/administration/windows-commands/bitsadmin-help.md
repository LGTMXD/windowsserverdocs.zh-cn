# <a name="bitsadmin-help"></a>bitsadmin help

显示有关 bitsadmin 命令行参数和选项的帮助相关信息。

## <a name="syntax"></a>语法

```
bitsadmin /help | /?
```

## <a name="examples"></a>示例

检索命令行帮助。

```
bitsadmin /help
```
