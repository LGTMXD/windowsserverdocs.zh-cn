# <a name="nslookup-finger"></a>nslookup/finger

与当前设备上的 finger 服务器连接。

## <a name="syntax"></a>语法

```
finger [<username>] [{[>] <filename> | [>>] <filename>}]
```

### <a name="parameters"></a>参数

| 参数 | 说明 |
| --------- | ----------- |
| `<username>` | 指定要查找的用户的名称。 |
| `<filename>` | 指定用于保存输出的文件名。 您可以使用大于（ `>` ）和双大于（ `>>` ）字符以常规方式重定向输出。 |
| /? | 在命令提示符下显示帮助。 |
| /help | 在命令提示符下显示帮助。 |
