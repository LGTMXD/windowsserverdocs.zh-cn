# <a name="online"></a>联机



使脱机磁盘或卷处于联机状态。

## <a name="syntax"></a>语法

```
Offline disk
Offline volume
```

### <a name="parameters"></a>参数

|参数|描述|
|---------|-----------|
|[脱机磁盘](offline-disk.md)|将焦点移动到联机状态。|
|[脱机卷](offline-volume.md)|使处于焦点状态的脱机卷进入联机状态。|
