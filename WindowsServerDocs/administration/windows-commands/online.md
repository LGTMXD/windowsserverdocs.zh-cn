# <a name="online"></a>联机

使脱机磁盘或卷处于联机状态。

## <a name="syntax"></a>语法

```
online disk
online volume
```

### <a name="parameters"></a>参数

| 参数 | 描述 |
|--|--|
| [online disk](online-disk.md) | 将焦点移动到联机状态。 |
| [online volume](online-volume.md) | 使处于焦点状态的脱机卷进入联机状态。 |
