# <a name="mqtgsvc"></a>mqtgsvc

当触发器的规则评估为 true 时，监视传入消息的队列，并以可执行文件或 COM 组件的形式执行操作。 有关如何使用 "消息队列触发器" 服务的示例，请参阅[消息队列触发器](https://docs.microsoft.com/previous-versions/windows/desktop/legacy/ms703197(v=vs.85))。

## <a name="syntax"></a>语法

```
mqtgsvc.exe
```
