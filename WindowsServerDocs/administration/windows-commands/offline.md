# <a name="offline"></a>离线



使联机磁盘或卷进入脱机状态。

## <a name="syntax"></a>语法

```
Offline disk
Offline volume
```

### <a name="parameters"></a>参数

|参数|说明|
|---------|-----------|
|[脱机磁盘](offline-disk.md)|使联机磁盘具有焦点，使其进入脱机状态。|
|[脱机卷](offline-volume.md)|使具有焦点的联机卷进入脱机状态。|
