# <a name="perfmon"></a>perfmon

在特定独立模式下启动 Windows 可靠性和性能监视器。

## <a name="syntax"></a>语法

```
perfmon </res|report|rel|sys>
```

### <a name="parameters"></a>参数

|参数|描述|
|---------|-----------|
|/res|开始资源视图。|
|/report|启动系统诊断数据收集器集，并显示结果报表。|
|/rel|启动可靠性监视器。|
|/sys|启动性能监视器。|
