# <a name="nslookup-set-port"></a>nslookup set port

将默认的 TCP/UDP 域名系统（DNS）名称服务器端口更改为指定值。

## <a name="syntax"></a>语法

```
set port=<port>
```

### <a name="parameters"></a>参数

| 参数 | 说明 |
| ---------- | ---------- |
| `<port>` | 为默认的 TCP/UDP DNS 名称服务器端口指定新值。 默认端口为**53**。 |
| /? | 在命令提示符下显示帮助。 |
| /help | 在命令提示符下显示帮助。 |
