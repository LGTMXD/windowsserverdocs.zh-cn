# <a name="bitsadmin-cache-and-getexpirationtime"></a>bitsadmin 缓存和 getexpirationtime

检索缓存过期时间。

## <a name="syntax"></a>语法

```
bitsadmin /cache /getexpirationtime
```

## <a name="examples"></a><a name=BKMK_examples></a>示例

以下检索缓存过期时间。

```
C:\>bitsadmin /cache /getexpirationtime
```
