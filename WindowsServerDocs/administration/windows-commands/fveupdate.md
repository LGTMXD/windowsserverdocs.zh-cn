# <a name="fveupdate"></a>fveupdate



FveUpdate 是一个内部使用的工具，当计算机从 Windows Vista 或 Windows Server 2008 升级到 Windows 7 或 Windows Server 2008 R2 时，安装程序将运行该工具。 它将与 BitLocker 关联的元数据更新到最新版本。 此工具不能独立运行。
