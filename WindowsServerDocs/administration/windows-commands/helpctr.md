# <a name="helpctr"></a>helpctr

Helpctr 命令已弃用，并且在将来的 Windows 版本中不一定会受到支持。

此工具包含在 Windows Server 2003 中。 有关详细信息，请参阅[Helpctr](https://docs.microsoft.com/previous-versions/orphan-topics/ws.10/cc755821(v=ws.10))。
