# <a name="detail-disk"></a>detail disk

显示所选磁盘的属性和该磁盘上的卷。 在开始之前，你必须选择一个磁盘才能使此操作成功。 使用 "[选择磁盘](select-disk.md)" 命令选择磁盘，并将焦点移动到该磁盘。 如果选择 (VHD) 的虚拟硬盘，则此命令会将磁盘的总线类型显示为 "*虚拟*"。

## <a name="syntax"></a>语法

```
detail disk
```

## <a name="examples"></a>示例

若要查看所选磁盘的属性以及磁盘中的卷的相关信息，请键入：

```
detail disk
```
