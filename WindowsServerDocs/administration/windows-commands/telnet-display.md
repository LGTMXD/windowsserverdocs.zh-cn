# <a name="telnet-display"></a>telnet：显示

>适用于：Windows Server（半年频道）、Windows Server 2016、Windows Server 2012 R2、Windows Server 2012

显示操作参数。   

## <a name="syntax"></a>语法  
```  
d[isplay]  
```  
#### <a name="parameters"></a>参数  
无  
## <a name="examples"></a><a name=BKMK_Examples></a>示例  
显示操作参数。  
```  
d  
```
