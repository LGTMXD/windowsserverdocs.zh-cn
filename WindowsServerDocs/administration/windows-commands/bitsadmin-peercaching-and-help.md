# <a name="bitsadmin-peercaching-and-help"></a>bitsadmin peercaching 和 help

显示 **/peercaching**开关的命令行用法。

## <a name="syntax"></a>语法

```
bitsadmin /peercaching /help
```

## <a name="examples"></a>示例

显示 **/peercaching**开关的命令行帮助：

```
bitsadmin /peercaching /help
```
