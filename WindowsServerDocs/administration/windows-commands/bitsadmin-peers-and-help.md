# <a name="bitsadmin-peers-and-help"></a>bitsadmin 对等方和帮助

显示 **/peers**开关的命令行用法。

## <a name="syntax"></a>语法

```
bitsadmin /peers /help
```

## <a name="examples"></a><a name=BKMK_examples></a>示例

以下命令显示 **/peers**开关的命令行帮助。

```
C:\>bitsadmin /peers /help
```
