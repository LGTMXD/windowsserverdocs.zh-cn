# <a name="bitsadmin-cache-and-list"></a>bitsadmin cache 和 list

列出所有缓存条目。

## <a name="syntax"></a>语法

```
bitsadmin /cache /list [/verbose]
```

## <a name="examples"></a>示例

列出详细格式的所有缓存条目。

```
bitsadmin /cache /list /verbose
```
