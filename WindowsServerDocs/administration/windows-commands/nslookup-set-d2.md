# <a name="nslookup-set-d2"></a>nslookup set d2



启用或禁用穷举调试模式。 打印每个数据包的所有字段。

## <a name="syntax"></a>语法

```
set [no]d2
```

### <a name="parameters"></a>参数

| 参数 |                             描述                              |
|-----------|----------------------------------------------------------------------|
| **nod2**  | 关闭穷举调试模式。 默认语法为**nod2**。 |
|  **d2**   |                 启用穷举调试模式。                  |
|   {帮助   |                                  ?}                                  |
