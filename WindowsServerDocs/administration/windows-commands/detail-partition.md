# <a name="detail-partition"></a>detail partition

显示选中分区的属性。

## <a name="syntax"></a>语法

```
detail partition
```

## <a name="remarks"></a>备注

-   必须选择分区，此操作才能成功。 使用 "**选择分区**" 命令可选择分区，并将焦点移动到该分区。

## <a name="examples"></a><a name=BKMK_examples></a>示例

若要查看所选分区的属性，请键入：
```
detail partition
```
