# <a name="attributes"></a>attributes

显示、设置或清除磁盘或卷的属性。

## <a name="syntax"></a>语法

```
attributes disk
attributes volume
```

### <a name="parameters"></a>参数

| 参数 | 描述 |
| --------- | ----------- |
| [attributes disk](attributes-disk.md) | 显示、设置或清除磁盘的属性。 |
| [attributes volume](attributes-volume.md) | 显示、设置或清除卷的属性。 |
