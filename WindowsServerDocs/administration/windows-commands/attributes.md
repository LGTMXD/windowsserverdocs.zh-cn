# <a name="attributes"></a>特性

显示、设置或清除磁盘或卷的属性。

## <a name="syntax"></a>语法

```
Attributes disk
Attributes volume
```

### <a name="parameters"></a>参数

| 参数 | 说明 |
| --------- | ----------- |
| [属性磁盘](attributes-disk.md) | 显示、设置或清除磁盘的属性。 |
| [属性数量](attributes-volume.md) | 显示、设置或清除卷的属性。 |
