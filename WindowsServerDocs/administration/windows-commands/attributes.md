# <a name="attributes"></a>attributes

显示、设置或清除磁盘或卷的属性。

## <a name="syntax"></a>语法

```
attributes disk
attributes volume
```

### <a name="parameters"></a>参数

| 参数 | 描述 |
| --------- | ----------- |
| [属性磁盘](attributes-disk.md) | 显示、设置或清除磁盘的属性。 |
| [属性数量](attributes-volume.md) | 显示、设置或清除卷的属性。 |
