# <a name="bitsadmin-cache-and-clear"></a>bitsadmin 缓存并清除

清除本地缓存。

## <a name="syntax"></a>语法

```
bitsadmin /cache /clear
```

## <a name="examples"></a><a name=BKMK_examples></a>示例

下面的示例将清除本地缓存。

```
C:\>bitsadmin /cache /clear
```
