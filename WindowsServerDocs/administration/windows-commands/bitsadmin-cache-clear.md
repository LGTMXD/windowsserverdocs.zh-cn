# <a name="bitsadmin-cache-and-clear"></a>bitsadmin cache 和 clear

清除本地缓存。

## <a name="syntax"></a>语法

```
bitsadmin /cache /clear
```

## <a name="examples"></a>示例

清除本地缓存：

```
bitsadmin /cache /clear
```
