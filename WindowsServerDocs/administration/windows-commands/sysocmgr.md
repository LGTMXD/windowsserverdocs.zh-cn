Sysocmgr 已被弃用，且在未来版本的 Windows 中不能保证受支持。

此工具包含在 Windows Server 2003 中。 有关详细信息，请参阅[Sysocmgr](https://technet.microsoft.com/library/cc773290(v=ws.10).aspx)。
