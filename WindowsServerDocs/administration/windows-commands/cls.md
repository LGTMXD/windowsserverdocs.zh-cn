# <a name="cls"></a>cls

清除 "命令提示符" 窗口。

有关如何使用此命令的示例，请参阅[示例](#BKMK_examples)。

## <a name="syntax"></a>语法

```
cls
```

### <a name="parameters"></a>参数

|参数|说明|
|---------|-----------|
|/?|在命令提示符下显示帮助。|

## <a name="examples"></a><a name=BKMK_examples></a>示例

若要清除 "命令提示符" 窗口中显示的所有信息并返回到空白窗口，请键入：
```
cls 
```
