# <a name="cls"></a>cls

清除 "命令提示符" 窗口。

## <a name="syntax"></a>语法

```
cls
```

### <a name="parameters"></a>参数

| 参数 | 描述 |
| --------- | ----------- |
| /? | 在命令提示符下显示帮助。 |

## <a name="examples"></a>示例

若要清除 "命令提示符" 窗口中显示的所有信息并返回到空白窗口，请键入：

```
cls
```
