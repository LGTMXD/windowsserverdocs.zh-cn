# <a name="nslookup-set-all"></a>nslookup set all



打印配置设置的当前值。

## <a name="syntax"></a>语法

```
set all 
```

### <a name="parameters"></a>参数

| 参数 | 描述 |
|-----------|-------------|
|   {帮助   |     ?}      |

## <a name="remarks"></a>备注

-   **全部设置**还会打印有关默认服务器和计算机（即主机）的信息。
