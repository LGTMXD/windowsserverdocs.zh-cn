# <a name="nslookup-set-all"></a>nslookup set all

输出当前配置设置值，包括默认服务器和计算机（主机）。

## <a name="syntax"></a>语法

```
set all
```

### <a name="parameters"></a>参数

| 参数 | 说明 |
| --------- | ----------- |
| /? | 在命令提示符下显示帮助。 |
| /help | 在命令提示符下显示帮助。 |
