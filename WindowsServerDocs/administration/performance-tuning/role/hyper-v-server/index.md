# <a name="performance-tuning-hyper-v-servers"></a>Hyper-V 服务器的性能优化


Hyper-V 是 Windows Server 2016 中的虚拟化服务器角色。 通过对处理器、内存和 I/O 设备进行虚拟化，虚拟化服务器可以托管彼此隔离但共享基础硬件资源的多个虚拟机。 通过将各个服务器合并到单台计算机上，虚拟化可以提高资源利用率和能源效率，并降低服务器的操作和维护成本。 此外，虚拟机和管理 API 在对资源进行管理、对负载进行均衡以及对系统进行预配方面提供了更大的灵活性。
